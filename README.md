## Basic Setup
```java
TideScript tideScript = new TideScript();

       tideScript.disable();
```
### Basic Logging Functions
```java
tidescript.
```
## Features
> Simple and Efficient logging with Encryption and File logs
> Ability to identify your project runtime.
> Ability to create plugins.
